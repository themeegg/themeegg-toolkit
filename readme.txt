== Description ==

ThemeEgg ToolKit for official themes.

Get free support at https://themeegg.com/support-forum/

= Requirements =

* WordPress 4.5 or later.


= Contribute =

You can contribute to the source code in our [GitHub](https://github.com/themeegg/themeegg-toolkit/) page.

== Installation ==

1. Download the plugin and extract the files
2. Upload "themeegg-toolkit" to your wp-
content/plugins/directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Put shortcode to anywhere.
5. Also you can change attribute of shortcode as you needed. 

== Features ==
1. Demo content import for ThemeEgg's official themes
2. Manual demo content import for all themes
